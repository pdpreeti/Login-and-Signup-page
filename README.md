Login & Signup Page-

A simple and responsive Login and Signup Page created using HTML and CSS. This project demonstrates how to design a clean and user-friendly authentication interface for websites.

Project Overview-

This project includes a login form for existing users and a signup form for new users. The interface is designed with a modern layout and styled using CSS to improve user experience. It focuses on front-end design and form structure.

The goal of this project is to practice HTML form creation and CSS styling techniques while building a visually appealing UI.

Features-
Clean and modern UI design
Login and Signup forms
Styled input fields and buttons
Hover effects on buttons
Icons for input fields
Responsive layout
Smooth form transition design

Technologies Used-
HTML5 – for page structure
CSS3 – for styling and layout
Boxicons – for icons used in the form
